# Demo Credit Wallet MVP

An API for managing user wallets, transactions, and funds.

## Features

- Register users (Blacklist check)
- Fund wallet
- Transfer funds
- Withdraw funds

## Tech Stack

- Node.js (LTS)
- TypeScript
- Knex.js (ORM)
- MySQL
- Jest (Testing)

## Setup Instructions

1. Clone the repository
2. Run `npm install`
3. Configure `.env`
4. Run migrations: `knex migrate:latest`
5. Start the server: `npm run dev`
