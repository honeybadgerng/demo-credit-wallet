# Demo Credit Wallet MVP

An API for managing user wallets, transactions, and funds.

## Features

- Register users (Blacklist check)
- Fund wallet
- Transfer funds
- Withdraw funds

## Tech Stack

- Node.js (LTS)
- TypeScript
- Knex.js (ORM)
- MySQL
- Jest (Testing)
